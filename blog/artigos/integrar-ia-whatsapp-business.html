Como Integrar IA com WhatsApp Business: Tutorial Técnico

Igor Silveira 15 de Janeiro de 2025

Integrar inteligência artificial com WhatsApp Business pode parecer intimidador se você nunca mexeu com APIs. Mas a verdade é que qualquer pessoa com conhecimento básico de programação consegue fazer essa integração em algumas horas.

Neste tutorial técnico, vou te mostrar exatamente como conectar IA (ChatGPT, Claude, ou modelos open-source) com WhatsApp Business API. Você vai aprender desde a configuração inicial até ter um assistente inteligente conversando naturalmente com seus clientes.

Arquitetura da Integração: Como Funciona

Antes de colocar a mão no código, entenda a arquitetura completa:

Componentes principais:

  1. WhatsApp Business API — Recebe e envia mensagens
  2. Webhook Server — Seu servidor que processa mensagens
  3. IA/LLM — ChatGPT, Claude, ou outro modelo de linguagem
  4. Banco de Dados — Armazena contexto e histórico de conversas
  5. Base de Conhecimento — Informações que a IA usa para responder

Fluxo de mensagem:

  1. Cliente envia mensagem no WhatsApp
  2. WhatsApp API envia webhook para seu servidor
  3. Seu servidor extrai mensagem e busca contexto no banco
  4. Servidor envia mensagem + contexto para API da IA
  5. IA gera resposta baseada na mensagem e base de conhecimento
  6. Servidor envia resposta de volta via WhatsApp API
  7. Cliente recebe resposta instantaneamente

Requisitos Técnicos

1. WhatsApp Business API

Opção 1: Meta Cloud API (Gratuito)

  • Primeiras 1.000 conversas/mês gratuitas
  • Setup mais rápido
  • Limitações de recursos

Opção 2: BSP (Business Solution Provider)

  • 360Dialog, Twilio, Infobip
  • Mais recursos e flexibilidade
  • Custo: R$ 200-500/mês

2. Servidor/Hospedagem

Opções recomendadas:

  • Railway.app — R$ 0-20/mês, fácil deploy
  • Render.com — Gratuito para começar
  • AWS Lambda — Serverless, paga por uso
  • DigitalOcean — Droplet de US$ 5/mês
  • Heroku — Fácil mas mais caro

3. API de IA

Principais opções:

  • OpenAI (ChatGPT) — US$ 0,002/1K tokens (GPT-3.5) ou US$ 0,03/1K tokens (GPT-4)
  • Anthropic (Claude) — US$ 0,008/1K tokens (Haiku) ou US$ 0,024/1K tokens (Sonnet)
  • Google (Gemini) — Gratuito até limite, depois US$ 0,0005/1K chars
  • Open-source — Llama 3, Mistral (auto-hospedado, mais barato em volume)

4. Linguagem de Programação

Este tutorial usa Node.js (JavaScript), mas conceitos aplicam-se a Python, PHP, etc.

Passo 1: Configurar WhatsApp Business API

Usando Meta Cloud API

1. Criar conta Meta Business:

  1. Acesse business.facebook.com
  2. Crie conta comercial
  3. Adicione número de telefone em "WhatsApp"
  4. Verifique número via SMS

2. Obter credenciais:

  1. Vá em "Configurações da API"
  2. Copie: Phone Number ID, WhatsApp Business Account ID, Access Token

3. Configurar webhook:

  1. Em "Configuração de Webhook", clique em "Editar"
  2. URL do Webhook: https://seu-servidor.com/webhook
  3. Token de Verificação: crie uma senha qualquer (ex: "meu-token-secreto-123")
  4. Marque "messages" em campos de webhook

Código: server.js

const express = require('express');
const bodyParser = require('body-parser');
require('dotenv').config();

const whatsappController = require('./controllers/whatsapp');

const app = express();
const PORT = process.env.PORT || 3000;

app.use(bodyParser.json());

// Verificação do webhook (Meta exige)
app.get('/webhook', (req, res) => {
  const mode = req.query['hub.mode'];
  const token = req.query['hub.verify_token'];
  const challenge = req.query['hub.challenge'];

  if (mode === 'subscribe' && token === process.env.VERIFY_TOKEN) {
    console.log('Webhook verificado!');
    res.status(200).send(challenge);
  } else {
    res.sendStatus(403);
  }
});

// Receber mensagens
app.post('/webhook', async (req, res) => {
  try {
    const body = req.body;

    if (body.object === 'whatsapp_business_account') {
      await whatsappController.handleIncomingMessage(body);
      res.sendStatus(200);
    } else {
      res.sendStatus(404);
    }
  } catch (error) {
    console.error('Erro no webhook:', error);
    res.sendStatus(500);
  }
});

app.listen(PORT, () => {
  console.log(`Servidor rodando na porta ${PORT}`);
});

Passo 3: Processar Mensagens do WhatsApp

Código: controllers/whatsapp.js

const axios = require('axios');
const aiController = require('./ai');
const db = require('../utils/database');

async function handleIncomingMessage(body) {
  const entry = body.entry[0];
  const changes = entry.changes[0];
  const value = changes.value;

  // Verificar se há mensagens
  if (!value.messages) return;

  const message = value.messages[0];
  const from = message.from; // Número do remetente
  const messageText = message.text?.body;
  const messageId = message.id;

  console.log(`Mensagem de ${from}: ${messageText}`);

  // Buscar contexto da conversa
  const context = await db.getConversationContext(from);

  // Enviar para IA
  const aiResponse = await aiController.generateResponse(
    messageText,
    context
  );

  // Salvar no banco
  await db.saveMessage(from, messageText, aiResponse);

  // Enviar resposta via WhatsApp
  await sendWhatsAppMessage(from, aiResponse);
}

async function sendWhatsAppMessage(to, text) {
  const url = `https://graph.facebook.com/v18.0/${process.env.WHATSAPP_PHONE_ID}/messages`;

  const data = {
    messaging_product: 'whatsapp',
    to: to,
    type: 'text',
    text: { body: text }
  };

  try {
    const response = await axios.post(url, data, {
      headers: {
        'Authorization': `Bearer ${process.env.WHATSAPP_TOKEN}`,
        'Content-Type': 'application/json'
      }
    });

    console.log('Mensagem enviada:', response.data);
    return response.data;
  } catch (error) {
    console.error('Erro ao enviar mensagem:', error.response?.data);
    throw error;
  }
}

module.exports = { handleIncomingMessage, sendWhatsAppMessage };

Passo 4: Integrar com IA (OpenAI/ChatGPT)

Código: controllers/ai.js

const OpenAI = require('openai');
const knowledge = require('../knowledge/base.json');

const openai = new OpenAI({
  apiKey: process.env.OPENAI_API_KEY
});

async function generateResponse(userMessage, context = []) {
  // System prompt define comportamento da IA
  const systemPrompt = `Você é um assistente virtual da empresa XYZ.
Seja cordial, objetivo e sempre ofereça soluções.

Informações da empresa:
${JSON.stringify(knowledge, null, 2)}

Regras:
1. Sempre cumprimente na primeira mensagem
2. Se não souber algo, seja honesto e ofereça transferir para humano
3. Use linguagem natural e amigável
4. Nunca invente informações
5. Seja proativo em oferecer produtos/serviços relevantes`;

  // Montar histórico da conversa
  const messages = [
    { role: 'system', content: systemPrompt },
    ...context,
    { role: 'user', content: userMessage }
  ];

  try {
    const completion = await openai.chat.completions.create({
      model: 'gpt-4-turbo-preview',
      messages: messages,
      max_tokens: 500,
      temperature: 0.7
    });

    const aiResponse = completion.choices[0].message.content;
    return aiResponse;

  } catch (error) {
    console.error('Erro na API OpenAI:', error);
    return 'Desculpe, estou com dificuldades técnicas. Um atendente humano vai te ajudar em breve.';
  }
}

module.exports = { generateResponse };

Passo 5: Gerenciar Contexto de Conversa

Código: utils/database.js

// Exemplo simples usando objeto em memória
// Em produção, use MongoDB, PostgreSQL, Redis, etc.

const conversations = {};

async function getConversationContext(phoneNumber) {
  if (!conversations[phoneNumber]) {
    conversations[phoneNumber] = [];
  }

  // Retorna últimas 10 mensagens para contexto
  return conversations[phoneNumber].slice(-10);
}

async function saveMessage(phoneNumber, userMessage, aiResponse) {
  if (!conversations[phoneNumber]) {
    conversations[phoneNumber] = [];
  }

  conversations[phoneNumber].push(
    { role: 'user', content: userMessage },
    { role: 'assistant', content: aiResponse }
  );

  // Limpar conversas antigas após 1000 mensagens
  if (conversations[phoneNumber].length > 1000) {
    conversations[phoneNumber] = conversations[phoneNumber].slice(-500);
  }
}

module.exports = { getConversationContext, saveMessage };

Troubleshooting: Problemas Comuns

Problema 1: Webhook Não Recebe Mensagens

Causas:

  • URL do webhook incorreta
  • Servidor não acessível publicamente
  • Token de verificação errado

Solução:

  • Verifique logs do servidor: pm2 logs
  • Teste URL manualmente: curl https://seu-servidor.com/webhook
  • Use ngrok para testar localmente: ngrok http 3000

Problema 2: IA Demora Muito para Responder

Causas:

  • Modelo GPT-4 é mais lento que GPT-3.5
  • Muitas mensagens no contexto

Solução:

  • Use GPT-3.5-turbo para respostas rápidas
  • Limite contexto a 10 mensagens
  • Adicione timeout na API da IA
  • Envie "digitando..." enquanto processa

Problema 3: Erro 401 na WhatsApp API

Causa: Access Token expirado ou inválido

Solução:

  • Gere novo Access Token no Meta Business
  • Use System User Token (não expira) em vez de Temporary Token

Problema 4: IA Inventa Informações

Causa: "Alucinação" do modelo de linguagem

Solução:

  • Reforce no system prompt: "NUNCA invente informações"
  • Use temperature baixo (0.3-0.5)
  • Valide respostas críticas com lógica de negócio
  • Use RAG (Retrieval-Augmented Generation) para respostas baseadas em documentos
